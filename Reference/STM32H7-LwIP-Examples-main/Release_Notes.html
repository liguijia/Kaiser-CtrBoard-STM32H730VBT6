Purpose

This examples shows how to configure and create simple projects for STM32H7 development boards, based on STM32CubeMX and LwIP stack middleware.

STM32H7 LwIP Examples are being delivered as-is, not necessarily supported by ST.

This example is based on the STM32CubeH7_v1.10.0 package. For further information about these components, see official package.

Update History

Main Changes

  • Added iperf measurement and TCP/IP settings tuned.
  • Added proper handling for multi-thread environment
